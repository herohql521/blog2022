# Angular DevTools

官方推荐 (opens new window)

# Path Intellisense

智能路径提示

# any-rule

常用正则，使用方式 F1 / 右键-正则大全 / @zz

# Auto Import

Typescript 导入分析提示

# Auto Rename Tag

自动同步修改闭合标签

# CSS Peek

可以html中点击 class/id 查看样式定义的位置，ctrl单击 / F12

# CSS Navigation

可以在html中选中class/id 右键-转到定义 来查看样式定义的位置， 右键-转到引用 来查看那些文件包含这个类名，开发的时候也可对已经定义的类目进行提示

# javascript console utils

console.log 生成和删除

生成：选中 cltr + shift + l生成选中的log , 不选中 生成空的 console.log('')

删除：cltr + shift + d 会删除当前页面所有的console.log()
